# **App Name**: Ração Certa

## Core Features:

- Listagem de Receitas: Exibe as receitas carregadas do arquivo receitas_local.json em uma lista.
- Seleção de Receita: Permite que o usuário selecione uma receita da lista para iniciar o processo de pesagem.
- Leitura de Peso da Balança: Lê os dados de peso da balança via porta serial em tempo real ou permite simulação via entrada de texto.
- Guia de Pesagem Ingrediente a Ingrediente: Apresenta o ingrediente atual, peso alvo e peso atual, guiando o usuário na adição correta dos ingredientes.
- Indicação Visual de Peso Atingido: Altera a cor do label de peso para verde quando o peso alvo é atingido, facilitando o processo de pesagem.
- Navegação entre Ingredientes: Permite avançar para o próximo ingrediente da receita ou voltar para o ingrediente anterior.
- Conclusão da Receita: Exibe uma mensagem de conclusão ao finalizar a pesagem de todos os ingredientes e retorna à tela principal.

## Style Guidelines:

- Primary color: Forest green (#228B22) for a natural and agricultural feel.
- Background color: Light beige (#F5F5DC) to provide a neutral and calming backdrop.
- Accent color: Earthy brown (#A0522D) to highlight important elements and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif, for a modern yet accessible look.
- Code font: 'Source Code Pro' for displaying simulated weight input when the serial port is not available.
- Use simple and clear icons representing each ingredient.
- Subtle transitions between ingredients in the weighing process.
